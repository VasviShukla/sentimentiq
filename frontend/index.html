REST API Endpoints

POST /predict — Single text

curl -X POST "https://sentimentiq-2.onrender.com/predict" \
  -H "Content-Type: application/json" \
  -d '{"text": "I love this product!"}'

POST /predict/batch — Multiple texts

curl -X POST "https://sentimentiq-2.onrender.com/predict/batch" \
  -H "Content-Type: application/json" \
  -d '{"texts": ["Great!", "Terrible.", "It was okay."]}'
